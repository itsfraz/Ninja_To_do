    Help & Information

    This is a powerful task management application. Here are some tips:

    • Use the input field to add new tasks.
    • Select a date and time for your tasks.
    • Categorize tasks with tags (Work, Personal, Shopping).
    • Sort tasks by due date, priority, or creation date.
    • Import and Export your tasks for backup or transfer.
    • Search for tasks using keywords.
    • Select multiple tasks to complete or delete them in bulk.
    • Drag and drop tasks to reorder them.
    • Click on a task to select/deselect it for bulk actions.
    • Completed tasks can be cleared using the 'Clean Up' button.
    • The calendar view helps you visualize your tasks by date.
    • Toggle between light and dark themes for comfortable viewing.
